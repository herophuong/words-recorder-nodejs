Ứng dụng thu âm từ đơn tiếng Việt (Node.js)
=========================

### I. Yêu cầu

1. [Node.js](https://nodejs.org) và [npm](https://www.npmjs.com/)
2. [sox](http://sox.sourceforge.net/)

### II. Cài đặt

```bash
    $ npm install
    $ npm install -g nodemon
```

### III. Sử dụng

1. Khởi động server

        $ npm start

2. Truy cập vào ứng dụng qua `http://localhost:3000`
